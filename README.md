# omarchy-minimal-waybar
A minimal waybar with cava and yeah, it adopts with your themes/wallpapers.

# Install dependencies for cava bar (using Arch-Linux)

```bash
  sudo yay -S cava
```

# Make the scripts executable
```bash
   chmod +x cava.sh
```
```bash
   chmod +x todo.sh
```
